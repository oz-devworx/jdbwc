Page Details:

Database core implementation.

Here we check for the type of database the user is asking to use and load the appropriate functions file for the db-type. Valid choices are mySql or postgreSql. MySql is the recommended database type. If you have the mysqli extesion enabled, you will have full acess to the JDBWC implementation.

A wcdb_exec_batch( mixed $query, [ $getGeneratedKeys = false])

wcdb_exec_batch() Process a single batch of sql queries.

Can be INSERT, UPDATE, DELETE, EXECUTE. $query should use complete inserts. Multiple rows per statement is ok: EG: INSERT INTO xyz (cell_01, cell_02, cell_03) VALUES ('abc', 'def', 'ghi'), ('jk', 'lmn', 'opq'), etc;




Tags:

return:  complex resultset bundle containing the rows affected by each query as a comma seperated list of integers on the first line. The rest of the resturn string is a set of double-newline seperated .csv tables produced by each select query in the batch.
