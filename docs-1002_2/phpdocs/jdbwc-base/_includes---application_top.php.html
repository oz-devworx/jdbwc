Page Details:

Remote JDBC Driver to local-database gateway.

This script is designed to communicate with a remote desktop application. Its purpose is to handle database access requests from the remote desktop portion of this Driver.

Allows the use of a http or https connection to access databases remotely whilst still maintaining localhost restrictions on the database daemon.

Connections to this script are established using apache jakarta.commons for java.

Apache webserver Credentials are required to access this scripts parent folder. Additional private credentials are required to access the database.
