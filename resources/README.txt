DISTRIBUTION CONTENTS:
======================
jdbwc-lib               contains the driver with dependencies.

server-side-bundle.zip  contains the server side part of the driver.

jdbwctest.zip           contains a sample test file for testing the driver
                        and demonstrates how to start a new java.sql.Connection

Install-xxx.html        Read this before using the driver. It explains the
                        connection URL, URL-parameters and how to install
                        the server-side-bundle and setup the driver.
